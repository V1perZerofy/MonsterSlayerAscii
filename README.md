# MonsterSlayerAscii

Ein Konsolen-Actionspiel in ASCII-Art, bei dem du als Held gegen ein Monster kämpfst! Die Steuerung erfolgt per Mausklick auf die farbigen Menüpunkte.

## Features

- Rundenbasierter Kampf: Angriff, Heilung, Spezialangriff (mit Cooldown), Aufgabe
- ASCII-Art-Logo und farbige Lebensbalken
- Mausunterstützung im Konsolenfenster (nur Windows)
- Zufallsbasierte Schadens- und Heilwerte
- Sieg- und Niederlagenbedingungen

## Steuerung

- Klicke mit der Maus auf die gewünschte Aktion im Menü:
  - **[A]TTACK**: Standardangriff
  - **[E]XTENDED ATTACK**: Starker Angriff (mit Cooldown)
  - **[H]EAL**: Heilen (nur wenn nicht voll geheilt)
  - **[S]URRENDER**: Aufgabe

## Voraussetzungen

- .NET 9.0 SDK
- Windows-Betriebssystem (für Mausunterstützung in der Konsole)
